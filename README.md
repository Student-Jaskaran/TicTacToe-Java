# ❌⭕ Tic-Tac-Toe – Java

A simple, clean Java Swing implementation of the classic two-player Tic-Tac-Toe game.

## 🎮 Features
- Two-player mode
- Instant win/tie detection
- Lightweight, intuitive GUI

## 🛠 Tech Stack
- Java
- Java Swing

## 🚀 Run
Open the project in IntelliJ IDEA and run `OX_Game.java`.
